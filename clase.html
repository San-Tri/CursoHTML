Componentes de HTML

Al iniciar una archivo HMTL en un editor de codigo nos muestra una una lista de etiquetas ¿que es lo que nos dice cada una de ellas?

!DOCTYPE html

nos dice que es un DOCUENTO TIPO HTML

html lang="es"

Sirve para especificar que el idioma del contenido html sera español, puedes cambiar las especificaciones del valor "idioma" usando el atributo lang ejm:
lang="es-MX" ara que el idioma sea español de Megico
lang="es-PN" ara que sea el español de Péru

meta charset="UTF-8"

esta etiqueta se utiliza para especificar la codificación de caracteres utilizada en el documento HTML
