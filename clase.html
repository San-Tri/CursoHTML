clase de HTML

Estructura basica del lenguaje HTML5

Al iniciar una archivo HMTL en un editor de codigo nos muestra una una lista de etiquetas ¿que es lo que nos indica cada una de ellas?

  • !DOCTYPE html

    Nos dice que es un DOCUENTO TIPO HTML.

  • html lang="en"

    Especifica que el idioma en el que se realizara este contenido html sera español, puedes cambiar las especificaciones del valor "idioma" usando el atributo lang ejm:
    lang="es" ara que el idioma sea español.
    lang="es-MX" ara que el idioma sea español de México.
    lang="es-PN" ara que sea el español de Péru.

  • head

    Esta etique que tiene una apertura (head) y un cierre (/head) es el encabezado la pagina, todas las etiquetas puestas entre las etiquetas head no se mostrar en la pagina web, pero contiene herraminetas que ayudara a que nuestra pagina sea ejecutada correctamente al abrirlo en el navegador, y lo veremos a continuación.

  • meta charset="UTF-8"

    Esta etiqueta se utiliza para especificar la codificación de caracteres utilizada en el documento HTML, eso permite que caracteres especiales, acentos y símbolos de diferentes idiomas se muestren correctamente en el navegador

  • meta name="viewport" content="width=device-width, initial-scale=1.0"

    Es una metaetiqueta que hace que la pagina web se renderise correctamente en una movil, la funcion de sus atributos son:
    • name="viewport": Indica que esta metaetiqueta está relacionada con la configuración de la vista del navegador
    • content="width=device-width, initial-scale=1.0": El atributo content especifica las propiedades de la ventana gráfica (ancho de pantalla) del navegador. los valores utilizados son:
      • width=device-width: Establece el ancho del dispositivo (pantalla) que se esta visualizando.
      • initial-scale=1.0: Establece el nivel de escala inicial para mostrar la página web al iniciar la pagina web.

  • title

    Es la etiqueta para ponerle un titulo a la pestaña de nuestra pagina web.
